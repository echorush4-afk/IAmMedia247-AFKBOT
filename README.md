# Crusty AFK Bot (Minecraft)

Discord bot + backend שמאפשר לכל משתמש להפעיל בוט AFK אישי בשרת Minecraft offline/cracked.

## תכונות
- פאנל שליטה עם כפתורים בדיסקורד
- תמיכה ב-/login אוטומטי
- רק חשבונות offline
- אוטומטי reconnect

## התקנה
1. `npm install`
2. העתק `.env.example` → `.env` והוסף DISCORD_TOKEN
3. `npm start`

## העלאה ל-Render
- Background Worker
- Build: `npm install`
- Start: `npm start`
